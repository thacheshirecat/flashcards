HTML

HTML stands for Hypertext(link) Markup Language. A Markup Language is A system for adding distinct instructions for how text should be formatted.

Block Elements

HTML elements that create a "block" in the display by appearing on their own line.

Inline Elements

HTML elements that do not appear on their own line, but instead share a line with other elements.

Tags

HTML annotations which indicate how the text should be formatted.

Attributes

Additional information provided to an HTML tag. For instance, the href attribute in an "a" tag provides the URL a link should travel to.

Relative path

A path to a file within the project itself. Usually referring to the location provided in as the href attribute in an "a" tag.
